# GIPF
a web implementation of GIPF, an [abstract 2 player board game designed by Kris Burm](https://boardgamegeek.com/boardgame/527/gipf). 

[Read the rules here](https://www.boardspace.net/gipf/english/rules.htm)

### Upcoming Features

- ability to undo a turn
- advanced play mode with gipf pieces
- remote 2 player
