# GIPF

![game in play](https://cdn.discordapp.com/attachments/859729768077066300/1115294801814884392/image.png)

a web implementation of GIPF, an [abstract 2 player board game designed by Kris Burm](https://boardgamegeek.com/boardgame/527/gipf). Currently local 2p only.

[Read the rules here](https://www.boardspace.net/gipf/english/rules.htm)

### Upcoming Features

- undo a turn
- advanced play mode with gipf pieces
- remote 2 player
